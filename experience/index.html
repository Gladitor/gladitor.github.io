# ExcelHandler

ExcelHandler是一款非常实用的Excel工作簿和工作表处理工具，这款软件目前仅支持在Windows操作系统下运行。支持去重、比对、批量合并整合等。

# 功能介绍

  • 合并

支持将多个Excel工作簿合并到一个Excel工作簿

支持将多个Excel工作表和工作簿批量合并到一个工作表

支持将多个Excel工作表和工作簿混合合并到一个Excel工作簿

  • 拆分

按行数拆分到多个工作簿或多个工作表

  • 比对

2个文件比对交并差集

  • 去重

单文件指定列去重

多文件指定列去重

# 系统要求

  • Windows
  • .NET Framework4.5.2
