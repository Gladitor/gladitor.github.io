# fiddler+nox对android抓包实战

  • fiddler:v5.0
  • nox:6.3.0.2

# fiddler常规配置

  • https

Tools->Options->HTTPS

  • connections

Tools->Options->HTTPS

# nox常规配置

  • wlan

设置->WLAN->WiredSSID,长按点击左键

修改网络->高级选项

代理服务器主机名是PC上网网卡的ip,代理服务器端口是fiddler->HTTPS中设置的端口，示例使用默认的8888

  • 证书安装

打开浏览器->地址中输入代理IP:端口访问->点击连接下载

若能正常安装证书则配置成功，若安装失败也不要慌，当然安装失败会有问题
证书安装失败会造成fiddler中无法抓取android中https的请求

链家app为例,只抓到了http的请求

# 手动安装证书

导出fiddler根证书到PC端

nox中打开文件中转站->打开安卓文件夹->把PC上的FiddlerRoot.cer拖拽到nox中

设置->安全->从SD卡安装

输入名字，设置PIN或密码，提示安装成功

# 抓包

nox中访问app->查看fiddler

fiddler功能强大，还可以配置自定义过滤规则 原创手写欢迎留言!
